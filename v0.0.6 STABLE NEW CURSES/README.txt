Plik README do aplikacji/kodu Labirynt

Potrzebne dodatkowe biblioteki:
New Curses <= 5

Reczna Kompilacja:
g++ <plik>.cpp -o <nazwa_wyjsciowego_pliku> -lncurses

Nalezy pominac <>

Dodatkowe uwagi:
Blad dolaczonego pliku wykonywalnego polegajacy na braku mozliwosci poruszania
Po ponownej kompilacji dziala poprawnie

Wszelkie zmiany nalezy zapisywac w osobnym folderze z dolaczona lista zmian

Edycja / Tworzenie poziomow
NIE MOZNA dawac w miejscu przezroczystej tekstury bialych znakow (spacja, tabulator, etc)
W tym celu nalezy uzywac dowolnego znaku innego niz "#"
